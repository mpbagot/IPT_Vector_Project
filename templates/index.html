About Arsenal:

Arsenal is a system designed to assist students in developing the required skills to create schematics for laser cutters. This system guides you through this process through two main tools, the tutorials, and the SVG Checker (further details below).

Tutorials:

Arsenal comes with a collection of pre-made tutorials, specifically targeted at assisting students with using the Adobe Illustrator software to develop vector graphics. These tutorials are split into topics, with each topic featuring tutorials targeted at a different stage of vector graphic development. Click the image above to view the tutorials...

SVG Checker:

In addition to the provided tutorials, Arsenal provides students with a specially designed SVG checker. This checker allows students to export their projects to an SVG formatted file, upload it, and be given a report about the issues with the schematic that will prevent it from correctly cutting with the laser cutter.
